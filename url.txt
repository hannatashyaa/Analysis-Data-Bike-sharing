Jika ingin menjalankan dashboard secara lokal:  
1. Pastikan semua library yang diperlukan sudah terinstal dengan menjalankan:  
   pip install -r requirements.txt

2. Masuk ke direktori dashboard:  
   cd dashboard

3. Jalankan aplikasi Streamlit:  
   streamlit run dashboard.py

4. Dashboard akan terbuka di browser. Jika tidak, akses melalui:  
   http://localhost:8501
